# UI Design Guidelines

## Color Palette
- Primary: Blue (blue-600)
- Success: Emerald (emerald-500)
- Warning: Amber (amber-500)
- Danger: Red (red-500)
- Neutral: Slate (slate-50 to slate-900)

## Typography
- Headings: font-semibold, tracking-tight
- Body: text-sm to text-base
- Labels: text-xs uppercase tracking-wide

## Spacing
- Base unit: 4px (Tailwind default)
- Cards: p-5 or p-6
- Tables: px-4 py-3 per cell
- Forms: gap-4 between fields

## Components
- Buttons: rounded-md, px-4 py-2
- Inputs: rounded-md, border-gray-200
- Cards: rounded-lg, border, shadow-sm
- Tables: rounded-lg, overflow-hidden

## Responsive
- Mobile-first approach
- Breakpoints: sm(640), md(768), lg(1024), xl(1280)
- Touch targets: minimum 44x44px

## Accessibility
- Focus rings on all interactive elements
- Sufficient color contrast
- Meaningful icons with text labels
- Loading states for all async operations
